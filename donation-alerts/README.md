# Donation Alerts Overlay

Animated OBS overlay for donation alerts. Made by Cyn.

## Features
- Purple gradient popup alerts
- Animated entrance
- Shows donor name and amount

## Installation
1. Add Browser Source in OBS
2. Point to overlay.html file
3. Set dimensions: 1920x1080
4. Check "transparent background"
